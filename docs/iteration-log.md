# Two-To 功能迭代记录

本文档用于记录 Two-To 项目的功能规划、开发时间节点、版本变更与阶段状态。每次新增功能、调整范围、完成开发或上线发布后，都应在这里追加记录。

## 状态说明

- `规划中`：已经确定方向，但尚未进入设计或开发。
- `设计中`：正在梳理产品方案、交互、接口或技术实现。
- `开发中`：已经进入编码实现。
- `联调中`：前后端或相关模块正在对齐验证。
- `已完成`：功能已完成开发并通过基础验证。
- `已上线`：功能已发布到可访问环境。
- `暂缓`：功能暂时不推进，等待后续确认。
