# Two-To

> 人与宠物最好的关系，是双向适配，彼此相伴。

Two-To 是一个围绕养宠人、宠物，以及想要养宠物的人打造的宠物服务产品。项目会先以 Web 端形态启动，逐步覆盖宠物适配匹配、品种解读、宠物档案、健康照料、日常管理与成长记录等能力，帮助每一段人与宠物的相遇更理性，也让长期陪伴更从容。

## 项目故事

很多人向往萌宠带来的治愈与陪伴，但在决定饲养时往往十分纠结：不清楚不同品种宠物的习性性格，也无法判断其是否适配自身生活方式。盲目冲动的选择，往往会造成双向内耗，辜负一份纯粹的生命羁绊。

催生这个项目的初心，缘起生活里一只可爱的猫咪——图图。正是这只小生命的存在，让我真切窥见人与宠物之间最纯粹的共生状态：包容、慰藉、双向适配。也让我愈发笃定，养宠从来不是单方面的占有与取悦，而是两个独立生命，互相适配、彼此接纳的漫长过程。基于这份亲身感触，我萌生了打造 Two-To 的想法，希望搭建一座温柔的桥梁，让每一份喜爱都能找到最合适的归宿，让每一次相遇都源于理解与契合。

项目名称 Two-To，读音溯源“图图”，同时以两个简单的英文单词，承载着我对这款产品、对人与宠物关系的全部理解与初心。

## 名称解读

### Two：两两相依，双向羁绊

Two，代表两两相依，是人与萌宠之间独有的双向羁绊。

这份羁绊也串联起两段完整的旅程：第一程是相遇与相知。我们结合适配性测评、品种解析、习性科普，陪你慢慢剖析自身生活现状、饲养条件与性格偏好，全方位拆解各类宠物的脾气、习惯与特质，帮你跳出跟风饲养、盲目选择的误区，从容找到那个和自己同频合拍的伙伴。

当缘分落地，便开启第二程——相守与照料。往后的朝夕里，我们用一系列实用的功能工具，帮你打理养宠生活里的大小琐事，记录宠物成长轨迹、守护生理健康、规划日常饮食起居，用心守护这份来之不易的羁绊，完成从“遇见”到“相守”的完整蜕变。

### To：步履不停，走向长久

To，代表步履不停，向着长久的美好不断前行。

我们的初心不止停留在“帮助大家找到合适的宠物”这一步。以此为起点，我们会持续打磨、迭代与丰富功能版图，逐步覆盖宠物档案管理、健康监测、用药提醒、日常喂养、成长记录等养宠全场景需求。把养宠过程中细碎、繁琐的事务一一梳理简化，不断扩充更多贴心、好用的能力，从单次的精准匹配，延伸至数年的长久陪伴，让产品的价值从当下，走向无限长远的时光。

## 产品愿景

Two-To 从来不止是一款功能工具，更是一份用心酝酿的温暖期许。

依托真实的陪伴感悟，我们兼顾适配匹配与全周期养护，规避盲目饲养的弊端，简化养宠日常的各项难题。愿每一位心怀热爱的人，都能在这里邂逅契合自己的萌宠；愿我们全程相伴，守护每一段双向奔赴的缘分，让朝夕相伴的温柔，岁岁绵长。

## 功能方向

- 适配性测评：帮助用户了解自身生活节奏、居住条件、性格偏好与养宠预期。
- 品种解析：沉淀不同宠物品种的性格、习性、照护难度与适配场景。
- 宠物档案：记录宠物基础信息、成长节点、照片与重要事件。
- 健康照料：管理疫苗、驱虫、体重、用药、就诊等健康相关事项。
- 日常管理：覆盖喂养、清洁、提醒、开销、备忘等养宠日常事务。
- 成长记录：记录人与宠物共同生活里的重要瞬间。

## 技术方向

- 后端：Go
- 形态：Web 端优先
- 仓库：Monorepo，统一管理前端、后端与项目文档
